# 🧑‍💻 Random User Generator

A sleek, interactive, and fully responsive web application that fetches and displays random user profiles (including name, picture, email, and location) on the fly. This project was built to master asynchronous JavaScript, RESTful API integration, and dynamic DOM manipulation.

## 🚀 Live Demo
[Click here to see the project live](https://randomuser-blush.vercel.app/)

## ✨ Features
- **Instant Data Fetching:** Generates a completely new, randomized user profile with a single button click.
- **REST API Integration:** Seamlessly communicates with a third-party API using asynchronous `fetch/await`.
- **Clean & Modern UI:** A minimalist, aesthetic design built with pure CSS that looks great on all screen sizes.
- **Robust Error Handling:** Graceful fallbacks and UI updates in case the API request fails or times out.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Deployment:** Vercel / GitHub Pages
